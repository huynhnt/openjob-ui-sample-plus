# Libraries & Frameworks

## Core Framework
- **Spring Boot Ecosystem**: Base framework for `openjob-spring-boot-samples` (`2.5.6`).
- **Spring Web**: Providing `@RestController` and routing.

## Dependency Services
- **Openjob Worker Core** (`1.0.7`): Raw Java library for job scheduling tasks.
- **Openjob Worker Spring Boot Starter**: Auto-configuration wrapper for Openjob in the Spring module.

## Utilities
- **Lombok**: Extensively used (`@Data`, `@AllArgsConstructor`, `@NoArgsConstructor`, `@Slf4j`).
- **SLF4J & Logback**: Default logging mechanisms across both modules.
- **Apache Commons Codec**: Used for basic encodings (e.g., Base64).
